📌 Project Overview

StudyFlow is a productivity web application designed to help students focus, track progress, and manage study sessions effectively.

This project demonstrates a responsive product landing page built using HTML, CSS, and JavaScript, along with Git/GitHub for version control.

---

 🚀 Features

⏱ Smart Timer

*Pomodoro timer (10–30 minutes)
*Pause, reset, and break timer
*Circular progress animation

📊 Progress Tracking

*To-do list with checkboxes
*Task completion tracking
*Pomodoro minutes tracker
*Daily streak system

🎯 Focus Mode

*Add current task
*Ambient background sounds
*Fullscreen focus mode
*Motivational quotes

📩 Contact Page

*Send feedback via email
*Clickable phone, Facebook, Instagram links

---

🛠 Technologies Used

*HTML5 (Semantic Structure)
*CSS3 (Flexbox, Responsive Design, Dark Mode)
*JavaScript (DOM, Events, LocalStorage)
*Git & GitHub (Version Control)
